JS：屏幕公告滚动（向上）效果

描述：

给用户的浏览器上一个可见区域，可见区域内只展示若干条数据，鼠标不在可见区域内，内容向上自动滚动，鼠标放在可见区域内，数据停止滚动，而且数据的滚动是可以循环的

准备：

CSS属性:

overflow:hidden 表示超出该容器以外的内容会被隐藏

JS相关知识：

  • onmouseover:鼠标悬浮事件
  • onmouseout: 鼠标停留事件

  • scrollTop:

    我的理解就是相当于摄影里面帧的概念，其实向上滚动并不是真正的滚动，只是像视频一样一帧一个图片那样。当scrollTop=0时，就是视频的第1帧

  • setInterval()定义和用法

    `setInterval(code, 周期);`

 或

 `setInterval(function, 周期, [param1, param2, ...可选]);`

`setInterval()` 方法可按照指定的周期（以毫秒计）来调用**函数**或**计算表达式**。

`setInterval()` 方法会不停地调用函数，直到 `clearInterval()` 被调用或窗口被关闭。由 `setInterval()` 返回的 ID 值可用作` clearInterval()` 方法的参数。

*提示： 如果你只想执行一次可以使用 `setTimeout()` 方法。*

  • clearInterval() 定义和用法

    `clearInterval()` 方法可取消由 `setInterval()` 函数设定的定时执行操作。

 `clearInterval()` 方法的参数必须是由 `setInterval()` 返回的 ID 值。

    注意: 要使用 clearInterval()方法, 在创建执行定时操作时要使用全局变量
